# Engineer AgentKit

A comprehensive AgentKit for software engineering tasks, including planning, architecture, coding, and testing.

## Features

- **Multi-mode operation**: Plan, Architect, Code, and Test modes
- **Structured workflows**: Template-driven development process
- **Best practices**: Built-in coding standards and validation
- **MCP integration**: Model Context Protocol support

## Modes

### Plan Mode
- Requirements analysis
- Task breakdown
- Timeline estimation

### Architect Mode
- System design
- Architecture decisions
- Technical specifications

### Code Mode
- Implementation guidelines
- Code quality standards
- Review processes

### Test Mode
- Test strategy
- Quality assurance
- Validation procedures

## Usage

After adding this kit with `agentkit add engineer`, your AI assistant will have access to structured engineering workflows and best practices.

## Files Included

- `main-agent.md` - Core agent configuration
- `main-rule.md` - Primary engineering rules
- `rules/modes/` - Mode-specific rules
- `templates/` - Planning and PRD templates
- `mcp.json` - MCP configuration
